VSCode安装教程（有手就行）

2024-01-25 09:27:00

1、准备

所谓万事开头难，而这第一难就是下载一个编辑器

但是，如果你有一个非常好的编辑器，那么，你就可以跳过这一步

那要怎么挑选一个好的编辑器呢？

这里推荐一下VSCode

他没有广告，并且界面简洁易用

那么首先，你就要上官网下载你的VSCode

https://code.visualstudio.com/

然后点击箭头指的这个按钮，不要点击按钮的向下小箭头

如果你是Mac或者Linux用户，那你就需要点击右上角的Dowload了

然后我们再根据自己所处的系统来选择相对应的安装包

等待下载安装包完成

然后在下载列表中右键打开安装包

见到欢迎界面，点击下一步

然后见到同意协议，点击我同意此协议，再点击下一步

来到了选择路径窗口，选择安装位置，单击浏览按需选择自己要设置的安装路径位置。然后点击下一步即可

注意：为了后期的稳定性，切勿使用中文路径

出现了创建开始菜单，不理他，直接点击下一步

出现了附加任务窗口，按需选择自己需要的附加任务，这里我推荐将其他中的所有选项都勾选上，对于创建桌面方式则按需选择是否勾选

设置完毕最后单击下一步

注意：添加到PATH一定要勾选，很重要

确认自己的设置无误后，点击安装即可,等待安装完成

如果你看到如下界面，那么恭喜你，你的VSCode安装完成。

点击完成，如果勾选运行VSCode将会弹出编辑器页面

注：这里的编辑器背景是通过插件来获得的效果，初始VSCode跟随系统深浅模式，背景为日本无条件投降1945年8月15日正午

你会发现，你启动的编辑器和图片上的中文不一样，是英文，为什么是英语呢？

因为当时的开发者落下了这个功能

其实不是，因为VSCode是一个轻量化的编辑器，所以默认会下载英文版来节省空间

那要如何改成中文呢？

答案是：加官方扩展扩展！！！

首先点击左侧的栏中这个按钮

然后输入Chinese (Simplified) (简体中文) Language Pack for Visual Studio Code

出现了几个标签，红框框出来的就是我们要找的🌏中文插件

点击标签右下角的Install

等他安装完，会弹出来一个弹窗，点击Restart，会重启VSCode，我们就安装好VSCode啦！
